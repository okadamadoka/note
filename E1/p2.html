結構使うやつコード

js

classList(クエリセレクター)
  • remove()メソッド - 指定したクラス名を削除する
  • add()メソッド - 指定したクラス名を追加する

css

  • z-index: 0;...優先度指定、数値が多いと前に行く
  • position: fixed;...固定表示
  • display: block;...ブロック要素にする
  • opacity: 0;...透明度
    • 1...不透明
    • .5...50%透明
    • 0...完全に透明
